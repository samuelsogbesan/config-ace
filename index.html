What is a Config File?

A config file also known as "autoexec" is a file that Counter-Strike: Global Offensive players use to store their CS:GO custom settings.

You can use it to setup basic things such as your crosshair, fps limit and senstivity settings, but also more complicating things like viewmodel, buy binds or radar.

You can even setup configurations for particular game modes for example a 128 tick 1v1 configuration.

The file extension for a config file is .cfg

Binding a Key to a Command

  1. Select a key from the onscreen keyboard.
  2. Choose any command from the command menu (e.g. cl_crosshair_style).
  3. Optionally, provide a value you want this command to take (e.g. 3)
  4. Click the button (or the Enter key) to save.
You can update any bind by selecting the key it is bound to and clicking on it again.

You can delete all binds using the ‚ùå key, or delete a specific bind by clicking the delete button in the command menu.

How to use a .cfg file

For most people, it will be as simple as moving the file to the directory where Counter-Strike: Global Offensive is installed.

The path may look something like: C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg

You can now reference your config file by name in the console, or you can add it to your CS:GO launch options on Steam.
